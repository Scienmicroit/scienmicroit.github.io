破解密码——利用 Windows PE 操作系统破解 Windows PIN

access_time 2020 年 5 月 26 日
by   Scienmicroit
chat_bubble 1

在此方法无效时可以利用该方法。

案例：利用 Windows PE 操作系统破解 Windows 10 的 PIN。

步骤

1. 开启计算机，下载安装 Windows PE 操作系统的制作工具（例如老毛桃、深度、大白菜等）后并打开，将空白 U 盘或光盘插入计算机，单击类似于 “ 一键制作 ” 的按钮，将 Windows PE 操作系统拷贝到空白 U 盘或光盘。
2. 设置你的 Windows PIN（即 Windows 密码；最好将 PIN 设置复杂一些，原因是为了突出显示最后破解成功效果），然后关机。
3. 在关机状态下插入带有 Windows PE 操作系统的 U 盘或光盘，开启计算机并立即按 F2 键（也可能是按 Delete 键或 ESC 键或其他键，不同计算机进入 BIOS 的方法可能不同，具体操作请咨询售后）进入 BIOS ，将带有 Windows PE 操作系统的 U 盘或光盘设置启动顺序为第一位（具体可根据 BIOS 界面的操作指导进行操作）并保存。
4. 重新启动计算机，进入 Windows PE 操作系统或 Windows PE 菜单，并使用其中的破解密码程序进行破解 Windows PIN。
5. Windows PE 操作系统是一个独立的操作系统，可以利用它来修改计算机系统中 C:\windows\system32\config\SAM 该目录下的 SAM 文件。 SAM 是 Windows PIN 的存储文件，修改后即可改变原 PIN 。
注： Windows PE 操作系统版本越高，可破解的 Windows 版本可能就越高。
